# Project Checklist

## Core Features Required

### Board Management
- [ ] Create multiple boards  
- [ ] Name boards  
- [ ] Rename boards  
- [ ] Delete boards  
- [ ] Switch between boards  

### Lists within Boards
- [ ] Add new lists to a board  
- [ ] Rename lists  
- [ ] Delete lists  
- [ ] Reorder lists (optional)  

### Card / Task Management
- [ ] Add cards to lists  
- [ ] Edit card content  
- [ ] Delete cards  
- [ ] Add card description  
- [ ] Add card due date  

### Drag and Drop
- [ ] Drag cards between lists  
- [ ] Show visual feedback during drag  
- [ ] Update card position in state/data  
- [ ] Prevent invalid drops  

### Data Persistence
- [ ] Save data to `localStorage`  
- [ ] Load data on page refresh  
- [ ] Auto-save on changes  

### Additional Features
- [ ] Search across all cards  
- [ ] Filter cards by keyword  
- [ ] Add labels/tags to cards  
- [ ] Show due date indicators  
- [ ] Archive completed tasks

## Technical Requirements
- [ ] Use ES6 classes for components  
- [ ] Implement event delegation for dynamic content  
- [ ] Use custom events for updates  
- [ ] Apply a state management pattern  
- [ ] Implement drag-and-drop logic
